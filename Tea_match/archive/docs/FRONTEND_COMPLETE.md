## 🛠️ 技术细节

### 后端架构

```
Flask 服务器 (app.py)
  ├─ GET / → 返回前端 HTML
  ├─ POST /api/recommend → 处理推荐请求
  │  ├─ 理解诉求 (understand_query.py)
  │  ├─ 澄清模糊诉求 (disambiguate_complaint.py)
  │  ├─ 搜索茶饮 (search_vector_store.py)
  │  ├─ 排序并取前 3 个
  │  └─ 生成推荐原因
  └─ CORS 支持跨域请求
```

### 前端特性

- ✅ 原生 HTML/CSS/JavaScript（无框架）
- ✅ 实时输入提交（Enter 键或按钮）
- ✅ 异步 API 调用
- ✅ 加载动画
- ✅ 错误提示
- ✅ 响应式设计
- ✅ 悬停交互效果
