# 交互式前端使用指南

## 🚀 快速开始

### 启动前端服务

```bash
cd D:\bayes\Tea_match
source .venv/Scripts/activate
python app.py
```

### 在浏览器中打开

打开浏览器，访问：

```
http://localhost:5000
```

### 使用流程

1. **输入诉求** - 在输入框中输入您的症状或疾病
   - 例如：`最近觉得心烦，卵巢囊肿，最近容易疲劳`
   - 支持逗号分隔多个诉求

2. **点击获取推荐** - 或直接按 Enter 键

3. **查看推荐结果** - 页面会显示：
   - 🥇 排名第1的茶饮（金色奖章）
   - 🥈 排名第2的茶饮（银色奖章）
   - 🥉 排名第3的茶饮（铜色奖章）

4. **查看推荐原因** - 每种茶饮都会显示：
   - 匹配症状数量
   - 详细推荐原因
   - 具体匹配的症状标签

---

## 💡 功能介绍

### 特点 1: 只推荐前 3 种茶饮

系统会分析用户的所有诉求，并只推荐最相关的 3 种茶饮：

- **第 1 名** - 匹配度最高
- **第 2 名** - 匹配度次高
- **第 3 名** - 匹配度第三

### 特点 2: 详细的推荐原因

每个推荐都包含：

```
✨ 推荐原因：与您的诉求『卵巢囊肿』『易疲劳』相匹配

🔍 匹配症状：
   [卵巢囊肿] [囊肿] [各类结节病] [女性癥瘕]
   [易疲劳] [乏力] [气虚] [改善心肺功能弱]
```

### 特点 3: 交互式设计

- ✅ 实时输入，无需刷新页面
- ✅ 按 Enter 键直接提交
- ✅ 美观的渐变色设计
- ✅ 悬停效果提升交互体验

## 🛠️ 技术栈

- **后端**: Flask + Python
- **前端**: HTML + CSS + JavaScript (原生，无框架依赖)
- **通信**: REST API + JSON
- **跨域**: CORS 支持
